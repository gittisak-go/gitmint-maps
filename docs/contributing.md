# การมีส่วนร่วม (Contributing)

ขอบคุณที่สนใจช่วยพัฒนาโปรเจคนี้! โปรดปฏิบัติตามแนวทางด้านล่าง:

- เปิด issue ก่อนเริ่มทำ feature หลักหรือการเปลี่ยนแปลงเชิงโครงสร้าง
- ตั้งชื่อ branch ตามรูปแบบ:
  - feature/<short-desc>
  - fix/<short-desc>
- เมื่อเสร็จ ให้เปิด Pull Request และเชื่อมโยงกับ issue ที่เกี่ยวข้อง
- ระบุขั้นตอนการทดสอบใน PR และรันทั้ง unit tests และ lint
- ปฏิบัติตาม CODE_OF_CONDUCT.md (ถ้ามี)

เทมเพลตสั้น ๆ สำหรับ PR:
- หัวข้อ:
- คำอธิบาย:
- วิธีทดสอบ:
- หมายเหตุ (breaking changes, migration steps)
